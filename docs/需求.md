      - 先对输入病历文本按中文句号/问号/分号分句，仅匹配同一句话内的实体对，避免跨句无效匹配

        2. 冲突处理策略：同一实体对匹配到多条规则时，**按优先级取最高优先级的单条规则**，不合并、不取多条；同一优先级内匹配到多条规则时，取规则定义顺序靠前的第一条
